# AutoApply AI

## Overview

AutoApply AI is an AI-powered Job Application Assistant that automates the process of analyzing job screenshots, calculating ATS compatibility, generating tailored resume suggestions, and creating personalized cold emails.

## Features

- OCR-based Job Description Extraction
- AI-powered JD Analysis
- ATS Score Calculation
- Matched & Missing Skills Identification
- ATS Resume Tailoring
- Personalized Cold Email Generation
- Google Sheets Tracking
- Gmail Draft Automation

## Tech Stack

- n8n
- Google Gemini AI
- OCR.Space API
- JavaScript
- Google Sheets
- Gmail API

## Workflow

Job Screenshot
→ OCR Extraction
→ JD Analysis
→ ATS Score Calculation
→ Resume Tailoring
→ Cold Email Generation
→ Google Sheets Logging
→ Gmail Draft Creation
